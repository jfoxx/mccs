# Multi-Site Manager Dashboard

A DA (Document Authoring) app that provides an AEM Multi-Site Manager Blueprint dashboard for previewing and publishing content across satellite sites.

## Features

- **Page Tree Navigation** — Browse your site's content via an expandable tree panel
- **Cross-Site Status** — See at a glance whether pages exist across all configured live copy sites
- **Bulk Preview & Publish** — Preview and publish selected pages across multiple sites in one operation
- **Activity Log** — Track all preview and publish operations with timestamped entries

## Configuration

Define your satellite sites in `/.da/msm.json` with `name` and `site` columns.
