# Icon Links Block

## Overview

The Icon Links block displays a horizontal row of icon + label links, typically used for category navigation (e.g., Movies, Lodging, Recreation, Fitness). Each item shows an icon above a text label and links to a destination.

## Usage

Structure your content in a table with two or three columns per row:

| Icon Links | |
|------------|---|
| [icon image] | [label text] |
| [icon image] | [label text] |
| [icon image] | [label text] |
| [icon image] | [label text] | [optional explicit href] |

- **Column 1**: The icon image (picture or img)
- **Column 2**: The label text — may be a link; if so, its href is used
- **Column 3** (optional): Explicit destination URL for the link

## Features

- Responsive layout: 4 per row (mobile), 6 per row (tablet), single row (desktop)
- When icons can’t all fit in one row, the list splits into exactly 2 rows with the smaller half on the bottom
- Hover lift effect
