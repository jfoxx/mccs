# Related Articles Block

Displays a list of related article links in the news template sidebar, matching the "Other Stories" pattern from MCCS source sites.

## Usage

Used by the importer when converting MCCS article pages. The "Other Stories" sidebar is transformed into this block with links to related articles.

## Structure

| Related Articles |
|------------------|
| [Article Title 1](url) |
| [Article Title 2](url) |
| [Article Title 3](url) |

## Styling

- Renders as a sidebar on news article pages (body.news)
- Uses DM Serif Text for the "Other Stories" heading
- Sticky positioning on desktop for easy access while scrolling
