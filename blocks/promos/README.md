# Promos Block

## Overview

The Promos block displays a row of promotional banner images, typically used for newsletter sign-up, social follow CTAs, or similar. Each promo is a full-width image that links to a destination.

## Usage

Structure your content in a table with one or two columns per row:

| Promos | |
|--------|---|
| [image] | [optional link URL] |
| [image] | [optional link URL] |
| [image] | [optional link URL] |

- **Column 1**: The promo image (picture or img)
- **Column 2** (optional): Destination URL — either a hyperlink or plain text path. If omitted, the image can be wrapped in a link in column 1.

Each row becomes a promotional card. Images are optimized at 800px width.

## Features

- Responsive layout: stacked on mobile, 3 across on tablet and desktop
- Gap between images for spacing
- Images use object-fit cover
